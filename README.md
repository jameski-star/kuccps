# KUCCPS - KCSE Cluster Points Checker

Welcome to the KUCCPS project! This platform is designed for KCSE students awaiting Form Four selection. Use this tool to check your cluster point eligibility for various university courses.

## Files

- `index.html` - Main application interface
- `Kuccps.xlsx` - Excel spreadsheet with course and cluster point data

## Description

This project provides KCSE candidates with information about university course eligibility based on cluster points. Students can use this platform to determine which courses they qualify for based on their grades.

## Usage Instructions

### For End Users:

1. **Accessing the Live Website:**
   - Visit the website at [https://your-domain.com] (replace with actual URL)
   - The site is hosted and ready for immediate use

2. **Local Access (Optional):**
   - If you have downloaded these files locally, simply double-click on `index.html` to open it in your web browser
   - For the best experience, it's recommended to access the live version online

3. **Accessing Additional Data:**
   - The `Kuccps.xlsx` file contains supplementary information
   - Double-click on `Kuccps.xlsx` to open it with your default spreadsheet application (Microsoft Excel, LibreOffice Calc, Google Sheets, etc.)

4. **Navigation:**
   - Use the menu items and links on the website to navigate between different sections
   - All interactive elements are fully functional on the live site
